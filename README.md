# PAGE
An app for decoding ambulance pager signals in New Zealand

Requirements
- Windows with XAMPP
- RTL2832U Based SDR
- SDR# Tuned to pager network (see radiowiki.org.nz)
- PDW with filters in place for ambulance callsigns and disabled FLEX-A decoding
